## Para Rodar a API

Na pasta backend

`npm install`
`docker-compose up -d`

### Criar Arquivo .env

DATABASE_URL="mysql://root:12345678@localhost:3306/to-do-list-api"
JWT_SECRET="123"

### Execute

Na pasta backend

`npm run start:dev`

## Para Rodar o FrontEnd

Na pasta frontend

`npm install`

### Execute

`npm run dev`
